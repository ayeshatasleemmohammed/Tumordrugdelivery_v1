# Targeted Drug Delivery v1

This is my first computational project in targeted drug delivery.

The aim of this project was to understand how nanoparticles behave while travelling through a blood vessel and interacting with a simplified tumor environment.

I developed this project as part of my journey into computational biomedical engineering. Although the model is highly simplified, it helped me understand several biological concepts and how they can be translated into code.

## What this project includes

- Brownian motion
- Blood flow
- Tumor targeting
- Immune clearance
- Drug release
- EPR effect
- pH-responsive nanoparticles
- Drug delivery efficiency

## Why I built this

Initially I wanted to understand how targeted drug delivery works from a computational perspective.

Instead of reading only theory, I decided to build a simulation from scratch and gradually add different biological mechanisms one by one.

This project represents my first attempt.

## Limitations

This is not a biologically accurate tumor model.

Many biological processes are represented using simplified assumptions and probabilistic models. The purpose of this project was learning rather than reproducing clinical or experimental results.

## What's next?

After completing this project, I realized that I was more interested in understanding the tumor itself than only simulating nanoparticles.

My next project will focus on building a computational model of the tumor microenvironment before introducing any drug delivery mechanisms.
